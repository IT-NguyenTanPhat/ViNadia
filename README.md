# ViNadia - VietNamese Social Media

ViNadia is a social media webapp that people can connect and share great memories to each other.

## A Fullstack Project

### Front-end

- React + TypeScript + Vite
- MUI
- Redux

### Back-end

- ExpressJS
- Mongoose
- Multer
- JWT

### Database

- MongoDB
